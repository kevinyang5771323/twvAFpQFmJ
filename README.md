# 员工管理系统 java1097

## 项目概述

员工管理系统是一款基于前后端分离架构的企业内部人员管理平台。该系统支持企业高效地进行员工信息管理，提高人事工作处理的便捷性。

## 技术栈

### 前端技术

- Vue
- Element-UI

### 后端技术

- Spring Boot
- MyBatis Plus

### 数据库

- MySQL

## 功能模块

- 员工信息管理
- 部门组织架构管理
- 权限与角色分配

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- 前端：现代浏览器
- 后端：Java 1.8 或更高版本
